# Makefile Generator
## _Wonderful makefile, everytime, everywhere_


Makefile generator is a bash script that allow you to generate clean, simple and handsome file for your own pleasure.

- launch a terminal
- execute the script
- ✨MAGIC ✨

## Features

- You can choose wich file you wan to add to your makefile (of course * still works)
- Libraries can of course be added when launching the script
- Clean command helps you to keep your folder orderly and fresh
- Run command do the work for you : clean, make and run




## Installation

Makefile generator DO NOT requires special modules to run.

Just download or clone the project
production node app
